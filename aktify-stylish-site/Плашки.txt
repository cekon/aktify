Текстовые плашки (бейджи) поверх картинок товаров делаются с помощью CSS-позиционирования .position: absoluteЧтобы плашка не улетала, контейнер картинки обязательно должен иметь свойство .position: relativeШаг 1. Добавьте разметку плашки в HTMLНайдите блок карточки товара в вашем HTML-файле. Оберните картинку в контейнер (если он еще не создан) и добавьте внутрь тег плашки:html<div class="product-card">
  <!-- Контейнер для картинки и плашки -->
  <div class="image-container">
    <span class="badge badge-hit">ХИТ</span>
    <span class="badge badge-discount">-15%</span>
    <img src="assets/images/box-close.jpg" alt="Товар" />
  </div>
  
  <div class="product-info">
    <!-- Название, цена, кнопка -->
  </div>
</div>
Use code with caution.Шаг 2. Добавьте стили в ваш CSS-файлОткройте ваш файл стилей ( или ) и добавьте в самый конец эти правила. Они зафиксируют текст поверх изображения:style.cssmain.csscss/* Контейнер картинки становится точкой отсчета для координат */
.image-container {
  position: relative;
  display: inline-block; /* или block, в зависимости от сетки */
  width: 100%;
  overflow: hidden;
}

/* Общие стили для всех плашек */
.badge {
  position: absolute;
  z-index: 10; /* Плашка всегда выше картинки */
  padding: 4px 8px;
  font-size: 12px;
  font-weight: bold;
  color: #ffffff;
  border-radius: 4px;
  text-transform: uppercase;
  letter-spacing: 0.5px;
}

/* Позиционирование и цвет для плашки "ХИТ" (слева вверху) */
.badge-hit {
  top: 10px;
  left: 10px;
  background-color: #ff3b30; /* Красный */
}

/* Позиционирование и цвет для плашки со скидкой (справа вверху) */
.badge-discount {
  top: 10px;
  right: 10px;
  background-color: #34c759; /* Зеленый */
}
Use code with caution.
