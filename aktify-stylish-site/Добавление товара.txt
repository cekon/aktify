Вам достаточно выполнить два шага: прописать параметры нового товара в коде сайта и синхронизировать его цену в коде Cloudflare Worker.Шаг 1. Добавление товара на сайт (assets/app.js)Откройте файл  и найдите массив  в самом начале файла. Каждый товар в нём — это отдельный блок в фигурных скобках .assets/app.jsproductsConfig{ ... }Чтобы добавить новый товар, поставьте запятую после последнего товара и вставьте новую структуру.Пример добавления второго товара ():product2JavaScriptconst productsConfig = {
  product1: {
    id: "product1",
    ozon_sku: "1695005157", // SKU, который мы отправляем в Ozon Pay
    title: "Головоломка Мудрец механики",
    price: 84000,
    desc: "Описание первого товара...",
    images: ["img/box-close.jpg", "img/video1.mp4"]
  }, // <--- ОБЯЗАТЕЛЬНО СТАВИМ ЗАПЯТУЮ ПЕРЕД НОВЫМ ТОВАРОМ
  
  product2: {
    id: "product2",
    ozon_sku: "987654321", // Замените на реальный SKU нового товара из Ozon
    title: "Новая интеллектуальная игра", // Название на сайте
    price: 1500, // Цена на сайте в рублях

    images: [
      "assets/images/new-puzzle-1.jpg", // Ссылка на главное фото
      "assets/images/new-puzzle-2.jpg"  // Дополнительное фото
    ]
  } // <--- У последнего товара запятую в конце ставить не нужно
};
Use code with caution.После этого новая карточка товара автоматически появится в галерее и модальном окне на фронтенде.Шаг 2. Защита цены в Cloudflare WorkerТак как ваш Cloudflare Worker работает по принципу безопасного каталога (чтобы пользователи не могли подменить цену в браузере), в него обязательно нужно внести точно такой же SKU и цену.Откройте ваш Cloudflare Worker, найдите внутри блок  и добавьте туда новый SKU:productsCatalogJavaScript// ЗАЩИЩЕННЫЙ КАТАЛОГ С ЦЕНАМИ ВНУТРИ ВОРКЕРА
const productsCatalog = {
  "1695005157": {
    name: "Головоломка Мудрец механики. Подарок мужчине. Классическая задача - крутить гайки в стесненных условиях",
    price: 84000
  }, // <--- СТАВИМ ЗАПЯТУЮ
  
  "987654321": { // Указываем ozon_sku нового товара

    price: 1500 // Цена в рублях (Воркер сам умножит её на 100 для копеек)
  }
};
